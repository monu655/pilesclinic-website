# 🏥 Piles Clinic Website

A professional static healthcare website designed for a Piles Treatment Clinic.
This project provides clear information about treatments, doctors, and medical services.

---

## 🌐 Live Demo

👉 https://monu655.github.io/pilesclinic-website/

---

## 📌 Project Overview

This website is built to provide patients with:

* Clinic information
* Specialist doctors details
* Available treatments
* Easy navigation for users

It is designed to be fast, responsive, and user-friendly.

---

## ✨ Features

* 🏠 Modern homepage design
* 👨‍⚕️ Doctors profiles section
* 🏥 Services: Piles, Fissure, Fistula, Laser treatment
* 📱 Mobile responsive layout
* ⚡ Lightweight and fast loading

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

---

## ⚠️ Important Note

This is a static website hosted on GitHub Pages.
Server-side features like PHP contact form are not supported.
